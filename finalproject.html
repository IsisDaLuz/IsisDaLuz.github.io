Final Project!

For our final project on CSC 106 class my partner, Alanis Irizarry, and I decided to code a trivia game called "Do you want to be a superstar?". In this trivia game you will be presented a question with four possible answers and you can only move on to the next question if you chose the correct answer. This trivia game as 15 questions and the goal is that you learn new facts that you can share with friends.
